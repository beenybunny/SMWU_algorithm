## 문제  
ch.6-8 순열 구하기

```
1부터 N까지 번호가 적힌 구슬이 있습니다.이 중 M개를 뽑아 일렬로 나열하는 방법을 모두 출력 합니다.

▣ 입력설명
첫 번째 줄에 자연수 N(3<=N<=10)과 M(3<=M<=N) 이 주어집니다.

▣ 출력설명
첫 번째 줄에 결과를 출력합니다. 맨 마지막 총 경우의 수를 출력합니다. 출력순서는 사전순으로 오름차순으로 출력합니다.

▣ 입력예제 1 32

▣ 출력예제 1 
1 2
1 3
2 1
2 3 
3 1 
3 2 
6
```

## 풀이
```python
def algorithm(self, n, m):
  def dfs(a, num):
    nonlocal res
    if num < m:
      for i in range(1, n+1):
        if str(i) in a:
          continue
        else:
          a.append(str(i))
          dfs(a, num + 1)
          a.pop()
    else:
      print(' '.join(a))
      res += 1
      return 

  res = 0
  dfs([], 0)
  print(res)
```
