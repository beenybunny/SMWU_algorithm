## 문제  
ch.2-5 정육면체 

```
두 개의 정 N면체와 정 M면체의 두 개의 주사위를 던져서 나올 수 있는 눈의 합 중 가장 확 률이 높은 숫자를 출력하는 프로그램을 작성하세요.
정답이 여러 개일 경우 오름차순으로 출력합니다.

▣ 입력설명
첫 번째 줄에는 자연수 N과 M이 주어집니다. N과 M은 4, 6, 8, 12, 20 중의 하나입니다.

▣ 출력설명
첫 번째 줄에 답을 출력합니다.

▣ 입력예제 1 46

▣ 출력예제 1 567
```

## 풀이
### 1
```python
def algorithm(self, dices):
  result = []
  total = 0
  min_value = min(dices)
  max_value = max(dices)
  for sum_value in range(min_value+1, sum(dices)):
    a = 0
    for i in range(1, min_value+1):
      if sum_value - i <= max_value:
        a += 1
    if a >= total:
      total = a
      result.append(sum_value)
    elif a < total:
      break
  return result
```

```python
def algorithm(self, n, m):
  result = [0]*(n+m+1)
  max = 0
  cnt = []
  for i in range(1, n+1):
    for j in range(1, m+1):
      result[i+j] += 1

  for k in range(n+m+1):
    if result[k] > max:
      max = result[k]
    elif result[k] < max:
      break

  for g in range(n+m+1):
    if result[g] == max:
      cnt.append(g)

  return cnt
```
